# Function: roundFloat()

> **roundFloat**(`value`, `digits`): `number`

Round a float value to `digits` decimal places. Examples:

```javascript
roundFloat(1.555, 2); // > 1.56
roundFloat(1.005, 2); // 1.01
roundFloat(2 / 3, 4); // 0.6667
```
